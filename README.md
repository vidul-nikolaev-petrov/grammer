# Правопис и пунктуация — Standalone Search Engine

Може да се ползва и офлайн, и [онлайн](https://vidul-nikolaev-petrov.github.io/grammer/).

* **Производителност**
  - Oптимизирано търсене ([binary search](https://en.wikipedia.org/wiki/Binary_search)) в паметта (~2MB).
* **Независимост**
  - Чист код (vanilla JS), позволяващ пълноценна работа офлайн.
