# Правопис и пунктуация — Standalone Search Engine

Може да се ползва и офлайн, и [онлайн](https://vidul-nikolaev-petrov.github.io/grammer/).

### Предимства:
* **Бързина:** Oптимизирано търсене чрез структура на индексите в паметта (1.2MB).
* **Независимост:** Чист JS код, което позволява пълноценна работа в офлайн режим.
